# QuickChat
 QuickChat website is built with the MERN stack. The website is designed for chatting.

## Link
https://quick-chat-8j1r.vercel.app/

## Functions/features of the website
* Login/Register
* Update Profile
* Chatting


## Technologies used


* HTML5
* CSS3
* JavaScript (ES6)
* GitHub
* React
* Tailwind
* MongoDB
* NodeJS
* ExpressJS
* Bcrypt
* JWT
* Validator
* React-icons
* Axios
